# AppDeliveryConfigurations
Responsible for DevOps configurations - added to other respiratories.

Use:  

  • cd AppDeliveryConfigurations

  • vagrant up 
  
  • vagrant ssh 
  
  • cd /project


Then the docker or docker-compose command can be used to build and run the app.
